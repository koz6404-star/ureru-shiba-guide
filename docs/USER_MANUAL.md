# 売れる柴ガイド - 利用マニュアル

フリマ出品補助ツールの使い方と設定についてまとめています。

---

## アプリで使うAPIキーについて

### このAPIキーとは？

**Anthropic Claude（クラウド）のAPIキー**です。  
説明文の自動生成機能で使われます。

### どこで取得する？

1. [console.anthropic.com](https://console.anthropic.com) にログイン（アカウントがない場合は新規登録）
2. 左メニュー **API Keys** をクリック
3. **Create Key** をクリックして新しいキーを作成
4. 表示されたキー（`sk-ant-api03-...` で始まる文字列）をコピー  
   ※このとき**一度しか表示されない**ので、必ずコピーして保存しておいてください

### アプリでの入力場所

1. アプリ画面上の **設定（⚙️）** をタップ
2. **Anthropic Claude APIキー** の欄にコピーしたキーを貼り付け
3. **保存** をタップ

## 料金のしくみ

### 基本ルール

**使う人が、使った分の料金を負担します。**

- あなたのAPIキーを使う → **あなたのAnthropicアカウントに課金される**
- パートナーAさんのAPIキーを使う → **Aさんのアカウントに課金される**

### パートナーに共有して使う場合

あなたのAPIキーをパートナーに入力してもらうと、**全員分の料金があなたのAnthropicアカウントにまとめて請求**されます。

- メリット: パートナーがAPIキーを用意しなくてよい
- 注意: 利用量が増えると、あなたの請求額が増える

料金を共有したい場合はあなたのAPIキーを、各自で負担したい場合は各自のAPIキーを使うよう案内してください。

### 料金の目安（参考）

- 説明文1件あたり: **数円〜数十円程度**（生成する文章量による）
- 詳細は [Anthropic 料金ページ](https://www.anthropic.com/pricing) で確認できます

## 会員登録（ログイン）について

### ログインするとできること

- APIキー・編集プリセット・保存済み出品が **端末間で同期**される
- スマホとPCで同じ設定・データが使える

### 新規登録の手順

1. 設定 → **新規登録**
2. メールアドレスとパスワードを入力
3. 登録後、メールで届く確認リンクをクリックして登録を完了

## トラブルシューティング

| 症状 | 確認・対処 |
|------|-----------|
| 説明文が生成されない | APIキーが正しく入力・保存されているか確認 |
| APIキーを忘れた | Anthropicコンソールで再発行し、再度入力 |
| ログインできない | メールの確認リンクを押して登録が完了しているか確認 |
